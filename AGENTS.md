# AGENTS.md

## 言語ルール

- このリポジトリに関する返答、レビューコメント、作業メモ、コミットメッセージ案、ログ文言、UI文言、エラーメッセージ、README、コードコメントなど、人間が読む文章は日本語で統一する。
- 外部API名、ライブラリ名、型名、関数名、識別子、設定キー、HTTPパスなどは英語のまま扱う。
- ユーザー向け文言は簡潔にし、実装意図が必要な箇所だけ補足する。

## 開発ルール

- Pythonは`uv`で管理し、`.python-version`と`pyproject.toml`に従う。
- ComfyUIは同梱しない。外部で起動しているComfyUI APIに接続する。
- ComfyUIが利用できない場合でも、stub画像生成でMVPが動作する状態を保つ。
- 画像生成モデルに日本語文字を描かせず、写植は後段のレンダリングで行う。
- 生成結果にはprompt、seed、backend、statusを残し、再生成可能性を優先する。
